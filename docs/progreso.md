# Resumen de progreso

**Fecha:** 20 de febrero de 2026

## Objetivo

Aplicación PWA para gestionar el balance personal registrando transacciones (gastos/ingresos), controlando saldos y recibiendo alertas al llegar a umbrales.

## Stack elegido

- React + TypeScript
- Firebase (base de datos)
- CSS

## Decisiones tomadas hasta ahora

- Target: PWA móvil (aplicación web instalable)
- Autenticación: Google Sign-In
- Funcionalidades MVP seleccionadas: múltiples cuentas, registro de transacciones, alertas/presupuestos, soporte offline y sincronización
